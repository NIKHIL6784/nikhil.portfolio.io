Project Overview

The OctoNyte Project is a university-led initiative to implement a RISC-V 32-bit processor using Chisel, supporting an in-order pipeline with multithreading capabilities.

My specific responsibility was focused on generating timed Register Transfer Level (RTL) designs from Chisel-generated Verilog using Yosys and performing synthesis optimizations for ASIC targets.
